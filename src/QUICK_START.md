## ğŸ› Troubleshooting

### Login Error: "Invalid login credentials"

**Cause**: You're trying to login without an account.

**Solution**: 
1. Use the Setup Page to create a demo account, OR
2. Register a new account manually

### Can't See My Data

**Cause**: You need to be logged in and have added consumers.

**Solution**:
1. Make sure you're logged in
2. Add a consumer number in your dashboard
3. Refresh the page

### Registration Failed

**Cause**: Email might already be registered or password is too short.

**Solution**:
1. Use a different email address
2. Ensure password is at least 6 characters
3. Check browser console for detailed error
